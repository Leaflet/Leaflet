A Leaflet plugin that integrates **DataTables**, **AJAX-loaded markers**, and **marker clustering**. 

- Load markers dynamically from AJAX/PHP backend  
- Display marker data in an interactive DataTable  
- Click table row → highlight corresponding marker on the map  
- Click marker → highlight corresponding table row  
- Filter/search DataTable → map updates dynamically  
- Supports marker clustering for large datasets  
- Fully configurable and modular Leaflet control
