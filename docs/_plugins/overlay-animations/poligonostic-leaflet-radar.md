### Poligonostic Leaflet Radar

This plugin displays animated weather radar frames, optional satellite imagery, and optional weather alerts using the LibreWXR API.

Install via NPM:
```
npm install poligonostic-leaflet-radar
```

Demo:
```
https://spoogle-search.github.io/poligonostic-leaflet-radar/examples/index.html
```
