This example makes use of Drawing.js, Drawing.LineUtils.js, Drawing.Polygon.js, and Drawing.Polyline.js. Use the buttons below to experiment.

The enable drawing button will enable drawing for the respective element. With drawing enabled, new vertices will be added in sequencial order to that element.

The enable editing button will enable edditing for that element. With editing enabled, markers will be placed at each vertex for that element. This allows a user to drag the marker and change the location of the vertex for the element. A single click on a marker will remove that marker and the vertex from the element.

Drawing and Editing modes can be enabled at the same time.
